# Expo Go 테스트 가이드

## 📱 Expo Go로 테스트하기

### 1. 사전 준비사항

#### ✅ 백엔드 서버 실행 확인
```bash
cd /Users/jeon/Desktop/Tishoo
./gradlew bootRun
```
- 서버가 `http://localhost:8080`에서 실행 중이어야 합니다.
- 터미널에 "Started TishooApplication" 메시지가 보여야 합니다.

#### ✅ 컴퓨터와 모바일 기기가 같은 Wi-Fi에 연결
- 컴퓨터와 스마트폰이 같은 Wi-Fi 네트워크에 연결되어 있어야 합니다.

#### ✅ 컴퓨터의 IP 주소 확인
```bash
# Mac
ifconfig | grep "inet " | grep -v 127.0.0.1

# Windows
ipconfig
```
- 현재 설정된 IP: `192.168.219.103`
- 다른 네트워크에서 테스트 시 IP 주소가 변경될 수 있습니다.

### 2. API 설정 확인

`src/config/api.js` 파일에서 IP 주소가 올바르게 설정되어 있는지 확인하세요:

```javascript
const IP_ADDRESS = '192.168.219.103'; // 컴퓨터의 실제 IP 주소
```

**⚠️ 중요:** 다른 Wi-Fi 네트워크로 변경하거나 IP 주소가 변경된 경우, 이 값을 업데이트해야 합니다.

### 3. Expo Go 실행

#### 방법 1: 터미널에서 실행
```bash
cd /Users/jeon/Desktop/TishooPub
npm start
```

#### 방법 2: QR 코드 스캔
1. 터미널에 QR 코드가 표시됩니다.
2. Expo Go 앱을 열고 QR 코드를 스캔합니다.
3. 앱이 로드됩니다.

### 4. 테스트 체크리스트

#### ✅ 네트워크 연결 확인
- Expo Go에서 앱이 로드되면, SearchScreen에서 API 호출이 자동으로 실행됩니다.
- 추천 멘토와 프로젝트 목록이 표시되어야 합니다.

#### ✅ API 연결 확인
1. **SearchScreen (탐색 탭)**
   - 추천 멘토 목록이 표시되는지 확인
   - 멘토 개설 프로젝트 목록이 표시되는지 확인
   - 로딩 중 표시가 나타났다가 사라지는지 확인

2. **에러 발생 시**
   - 네트워크 연결 확인
   - 백엔드 서버가 실행 중인지 확인
   - IP 주소가 올바른지 확인
   - 방화벽 설정 확인 (8080 포트 허용)

### 5. 문제 해결

#### 문제: "네트워크 연결을 확인해주세요" 에러
**해결 방법:**
1. 컴퓨터와 스마트폰이 같은 Wi-Fi에 연결되어 있는지 확인
2. 컴퓨터의 IP 주소를 다시 확인하고 `src/config/api.js`에 업데이트
3. 백엔드 서버가 실행 중인지 확인
4. 방화벽에서 8080 포트가 허용되어 있는지 확인

#### 문제: 데이터가 표시되지 않음
**해결 방법:**
1. 백엔드에 데이터가 있는지 확인 (멘토, 프로젝트)
2. 브라우저에서 `http://localhost:8080/api/mentors/recommended` 접속하여 데이터 확인
3. 콘솔 로그 확인 (Expo Go에서 개발자 메뉴 열기)

#### 문제: IP 주소가 변경됨
**해결 방법:**
1. 새로운 IP 주소 확인
2. `src/config/api.js`의 `IP_ADDRESS` 값 업데이트
3. 앱 재시작

### 6. 개발자 도구 사용

#### Expo Go에서 개발자 메뉴 열기
- **iOS**: 화면을 3번 탭하거나 흔들기
- **Android**: 기기를 흔들거나 메뉴 버튼 누르기

#### 콘솔 로그 확인
- 개발자 메뉴에서 "Debug Remote JS" 선택
- Chrome DevTools에서 콘솔 로그 확인 가능

### 7. 다음 단계

현재 연결된 기능:
- ✅ 추천 멘토 조회 (SearchScreen)

### 8. 참고사항

- **iOS 시뮬레이터**: `localhost` 사용 가능
- **Android 에뮬레이터**: `10.0.2.2` 사용
- **Expo Go (실제 기기)**: 컴퓨터의 IP 주소 사용 필요

현재 설정은 Expo Go (실제 기기)를 기준으로 되어 있습니다.
